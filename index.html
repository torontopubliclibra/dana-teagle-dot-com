The Pigeon Hole

HTML, CSS, jQuery, REST API

The Pigeon Hole is jQuery API project that I built for the Juno JavaScript course in July 2022.

This app demonstrates the globe-spanning urban habitats and biodiversity of my favourite bird, displaying photos of pigeons in different cities based on the's user selection or by random.

Functionally, the site makes an Ajax request to the Flickr API for the 10 most relevant photos in the selected or randomized city, then randomly selects one image, and adds that image to the page through DOM manipulation.

ListFlicks

HTML, SASS, JavaScript, REST API

ListFlicks is a vanilla JavaScript API project built with Serena Kang for the Juno Web Development bootcamp in the fall of 2022.

This app displays three movies based on the user input by making an Ajax request to a movie database API for a list of movies matching the user's query.

The site allows the user to open a pop-up modal to view more info about the movies (making a second API call using the selected movie's unique ID) and also to add these items to a list, where they can favourite, cross off, or remove movies as they make their way through the list.

Under the Sea

HTML, SASS, JavaScript

Under the Sea is a PSD conversion project that I built for the Juno Web Development bootcamp in late summer 2022.

The multi-page site uses HTML and SASS to render a complex design of overlapping sections and images, as well as interactive functionality.

I designed the site to feature multiple CSS breakpoints, allowing for seamless responsivity across devices, and as stretch goals, I added a functional hamburger menu, a scroll to top button, and some accordion sections using vanilla Javascript.
