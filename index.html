The Pigeon Hole

HTML, CSS, jQuery, REST API

A jQuery API project built for the Juno JavaScript course, this app displays photos of pigeons in different cities. On the user's selection of a city or on click of the random button, the app makes an Ajax request to the Flickr API for the 10 most relevant photos in the selected or randomized city, randomly selects one image, and adds that image to the page.

ListFlicks

HTML, SASS, JavaScript, REST API

A vanilla JavaScript API project built with Serena Kang for the Juno Web Development bootcamp, this app displays movies based on the user input. The app makes an Ajax request to a movie database API for movie IDs matching a query. The functionality allows the user to read more info about the movies (through a second API call) and to add these items to a list.
